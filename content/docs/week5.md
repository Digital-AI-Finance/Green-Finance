# Week 5: Project Finance

## Learning Goals

1. **Understand renewable energy project economics** including LCOE trends
2. **Apply project finance modeling** techniques (DSCR, IRR analysis)
3. **Evaluate green infrastructure investments** through case studies

## Topics Covered

- Levelized Cost of Energy (LCOE)
- Renewable capacity growth trends
- Project finance structures
- DSCR sensitivity analysis
- Cash flow modeling
- IRR sensitivity analysis
